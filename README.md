# Adryan Maha Putra – Developer Portfolio

Modern, futuristic personal portfolio website built with React + Vite.

## 🚀 Tech Stack

- **React** (JavaScript) + **Vite**
- **Framer Motion** for smooth animations
- **React Icons** for icons
- **Custom CSS** – dark futuristic theme with glassmorphism, neon glow, and cyberpunk elements

## 📦 Installation

```bash
npm install
```

## 🧪 Run Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🔨 Build for Production

```bash
npm run build
```

Output will be in the `dist/` directory.

## 🌐 Deploy to Vercel

1. Push to GitHub
2. Connect repo to [Vercel](https://vercel.com)
3. Set:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
4. Deploy

## ✏️ Update Portfolio Data

All portfolio content lives in a single file:

```
src/data/profileData.js
```

Edit this file to update:
- Profile info, social links
- Education, experiences
- Skills (categories & levels)
- Projects
- Research, leadership, certifications, awards
- Contact information

## 📁 Project Structure

```
src/
  components/     # All React components
  data/           # Portfolio data (profileData.js)
  styles/         # Global CSS
  App.jsx         # Main app
  main.jsx        # Entry point
public/
  assets/         # Project images, certificates
  Adryan-Maha-Putra-CV.pdf
```
